### 이벤트 핸들러 경우 
- Props인 경우
Props 이름을 정의할 때는 보통 onClick과 같이 on* 접두사를 지정합니다.

- Function Names의 경우
 on*을 handle*로 변경하여 handleClick과 같이 사용합니다.
>> ex) <MyComponent 
>        onAlertClick={this.handleAlertClick} 
>        onFormSubmit={this.handleFormSubmit} 
>     />
